Introduction

If you already have Bootstrap in your style sheet,skip this section.
To add Bootstrap CSS to your site, add the following line of code into the head section of your html file.
This makes use of a Content Delivery Network (CDN). 

                    
                    
                    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">

You can also download and use a local version of the CSS sheet. This is not recommended, but if you want to learn more here

                <div class="row">

                    </div>

Buttons

With bootstrap, you are able to create buttons of many styles without the hastle of making them look nice. This is written as follows: 
                    
                    <button type="button" class="btn btn-XXXX">Title

The different styles are displayed below with their names (what would replace XXXX in the code above).



Other Button Styles
Note: These can all be used with all above styles, primary is being used as an example.

Outline buttons 

                    
                    <button type="button" class="btn btn-outline-primary">Primary




Disabled buttons 

                    
                    <button type="button" class="btn btn-primary" disabled>Primary



Try pressing the button above!
Note: This does not prevent the button from being clicked at all costs; someone can always edit the html from their browser!

Buttons as a class 

                    
                    <a class="btn btn-primary" href="#" role="button">Link
                    <input class="btn btn-primary" type="submit" value="Submit">

Media

Bootstrap takes care of problems with displaying media objects alongside text.
Media heading
Text is formatted next to the image automatically.

Basic code for media formatting 

                <div class="media">
                    <img class="mr-3" src="https://assets-cdn.github.com/images/modules/logos_page/GitHub-Mark.png" alt="git octopus">
                    <div class="media-body">
                        <h5 class="mt-0">Media heading
                        Text is formatted next to the image automatically.
                    </div>
                </div>

Tables

Bootstrap provides easy customization for tables, which are designed to be opt-in.

col0 col1 col2 col3
1 1.1 1.2 1.3
2 2.1 2.2 2.3
3 3.1 3.2 3.3 


    
                <table class="table">
                    <thead class="thead-dark">
                        <tr>
                            <th scope="col">col1
                            <th scope="col">col2
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <th scope="row">1
                            <td>1.1
                        </tr>
                    </tbody>
                </table>
